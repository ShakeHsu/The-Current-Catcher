# Backtrader回测使用说明

## 文件说明

已为您创建了Backtrader版本的策略文件：
- **原PTrade策略**：`顺势而为+激进+做T.py`（保持不变）
- **Backtrader版本**：`顺势而为+激进+做T_backtrader.py`（用于本地回测）

## 环境要求

要在本地运行Backtrader回测，需要安装以下库：

```bash
pip install backtrader pandas numpy akshare matplotlib
```

## 运行回测

### 方法一：命令行运行

```bash
python 顺势而为+激进+做T_backtrader.py
```

### 方法二：在Python环境中运行

```python
exec(open('顺势而为+激进+做T_backtrader.py').read())
```

## 策略参数

Backtrader版本完整实现了原PTrade策略的所有逻辑：

### 买入条件
1. 前5天中有3天满足：收盘价>开盘价 且 收盘价>5日均线
2. 前5天中有3天满足：5日均线上升
3. 当日最低价 < 昨日收盘价（回踩确认）
4. 反弹确认、成交量过滤、资金管理
5. 当前价格不低于上次买入价格

### 分批止盈（满足任意一条卖出一半）
- 放量大涨（当日成交量 > 前5日平均成交量 × 1.5）
- 盈亏率 > 15%

### 清仓条件（满足任意一条清仓）
- 股价从最高点回撤5%
- 盈亏率 < -5%

### 做T条件
- 当日已买入，单日涨幅>5%且回落1%
- 或14:55涨幅>2%

## 输出结果

回测完成后会输出：
- 最终市值
- 总收益率
- 夏普比率
- 最大回撤
- 年化收益率
- 交易次数、盈利次数、亏损次数

## 注意事项

1. **数据获取**：Backtrader版本使用AKShare获取历史数据，需要网络连接
2. **时间范围**：当前设置回测时间为2025.12.1-2026.3.12，如需调整请修改代码中的日期参数
3. **代码差异**：Backtrader版本与PTrade版本在API调用上有差异，但核心逻辑完全一致
4. **结果对比**：Backtrader回测结果仅供参考，最终请以PTrade平台回测结果为准

## 常见问题

### Q: 为什么Backtrader结果和PTrade结果可能不同？
A: 可能原因：
- 数据源差异（AKShare vs PTrade数据）
- 交易时间处理差异
- 手续费、滑点计算方式差异
- 复权方式差异

### Q: 如何修改回测参数？
A: 在代码中修改`params`元组中的参数值，例如：
```python
params = (
    ('buy_amount', 20000),  # 修改为每次买入2万元
    ('max_cost', 300000),   # 修改最大持仓成本为30万
    # ... 其他参数
)
```

### Q: 可以回测其他股票吗？
A: 可以，修改`params`中的`security`参数和`ak.fund_etf_hist_em`中的代码：
```python
# 例如改为沪深300ETF
params = (
    ('security', '510300.SH'),
    # ...
)
# 修改数据获取
df = ak.fund_etf_hist_em(symbol="510300", ...)
```
